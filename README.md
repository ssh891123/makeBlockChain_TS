# makeBlockChain_TS

1. git clone //소스를 다운 받음
2. tsc --init //tsconfig.json 생성
3. tsc -w index.ts //index.ts을 index.js로 생성함
4. 새로운 터미널 창 열기
5. node index.js //실행하기
6. 터미날 창에 Block 배열정보가 나타나면 성공
